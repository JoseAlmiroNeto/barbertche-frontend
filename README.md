# BarberTche

Protótipo mobile em React Native/Expo para agendamento de barbearia premium.

## Rodando

```bash
npm install
npm run start
```

O app usa dados em memória para demonstrar os fluxos de cliente e administrador, incluindo bloqueio de horários ocupados, recorrências semanais, horários de funcionamento, datas fechadas e bloqueios manuais.
